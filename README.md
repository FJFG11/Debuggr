# Debuggr

Debuggr is a simple client/server executor for roblox developers for their games this is NOT a injector/exploits for roblox.

## Setup

1. Download the 'Debuggr V1.1.rbxm'
2. Insert the file into your game
3. Put 'MainModule' into ServerScriptService
4. Open the 'Whitelist' module (MainModule -> DebugExec -> WHITELIST)
5. Insert your userid into the table.

## Notes

- Using print, warn on the server with output a error.
- If you paste code with comments into the textbox the text colouring will break.
- Im working on improving and fixing debuggr.
