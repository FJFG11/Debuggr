# Debuggr

Debuggr is a simple client/server executor for roblox developers for their games this is NOT a injector/exploits for roblox.

# Setup

1. Insert the file into your game
2. Put 'MainModule' into ServerScriptService
3. Open the 'Whitelist' module (MainModule -> DebugExec -> WHITELIST)
4. Insert your userid into the table.
